Note:
Download the eudev-3.2.7.tar.gz from https://dev.gentoo.org/~blueness/eudev/eudev-3.2.7.tar.gz
The eudev-3.2.7 tool depends on the gperf tool during compilation. The gperf tool must be installed on the server.
Download website: http://ftp.gnu.org/pub/gnu/gperf/gperf-3.1.tar.gz
Installation description:
tar -xzf gperf-3.1.tar.gz
cd gperf-3.1
./configure
make
make install
