Due to the rectification of open-source tools, the following open-source tools do not provide source code packages. 
Please download them before compilation.
Download the libuuid-1.0.3.tar.gz from https://sourceforge.net/projects/libuuid/ 
Download the zlib-1.2.11.tar.gz from https://sourceforge.net/projects/libpng/files/zlib/1.2.11/zlib-1.2.11.tar.gz/download?use_mirror=nchc&download=
Download the mtd-utils-2.0.2.tar.bz2 from https://src.fedoraproject.org/repo/pkgs/mtd-utils/mtd-utils-2.0.2.tar.bz2/
Compile Tools by Makefile
