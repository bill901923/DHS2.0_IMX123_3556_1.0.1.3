Usage:

1. Run make in directory sample
;
2. Run 'insmod hi_ipcm.ko' in linux serial on board. You can find hi_ipcm.ko in osdrv;
3. Input './sample_sender' in linux serial, input 'sample_receiver' in liteos serial. then the message will be transfering from linux to liteos;
4. Input 'q',then the sample will quit.
